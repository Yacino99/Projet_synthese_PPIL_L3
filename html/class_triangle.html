double Triangle::calculerAire ( ) const
virtual

Méthode qui retourne l'aire du triangle.

La formule de Héron stipule que l'aire A d'un triangle de côtés de longueurs a , b et c est A = √ d ( d − a ) ( d − b ) ( d − c ) où d est le demi-périmètre du triangle ou la moitié de son périmètre.
Perimètre = somme des 3 cotes du triangle.

Returns
Un double
